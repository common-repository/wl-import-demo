== Description ==

== Installation ==

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `wl-import-demo.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `wl-import-demo.zip`
2. Extract the `wl-import-demo` directory to your computer
3. Upload the `wl-import-demo` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
